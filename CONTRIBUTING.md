# Contributing to Printverse

Thanks for your interest in contributing! To maintain quality and clarity, please follow these steps:

## How to Contribute

1. **Fork the repo**
2. **Create a new branch from `main`**
3. **Address open Issues or create a new Issue describing your proposed change**
4. **Commit code using clear messages referencing the related Issue**
5. **Open a Pull Request and link the Issue (Closes #X)**

## Guidelines

- Write clean, readable code (preferably with ESLint/prettier conventions!)
- Add/maintain tests for new/updated code
- Document public functions and modules
- Do NOT push directly to main
- Use standard Issue and PR templates
- Be respectful and collaborative

## Need Help?

Open an Issue with your question or proposal. Maintainers and the community are here to help!
